# terminal-tetris
Terminal Tetris for linux.
---
# Controls:
* Arrow keys to move:
  - Left, Right are obvious
  - Down - soft drop
  - Up - rotate
* Shift - hold
* Space - hard drop
* ^C to exit
# Dependencies:
If you wish to play, please install the following modules from PyPI:
* `pynput`
* `termcolor`
